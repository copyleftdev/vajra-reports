The Three Cost Tiers

Tier 1 — Cheap (2-12x) Effect.gen, Option, Either, basic pipes. The idiomatic path is only 2x overhead. If you stick to generators, Effect is nearly free.
Tier 2 — Moderate (12-50x) Schema validation, runSync, error handling, Chunks. Each runSync boots the full fiber runtime.
Tier 3 — Expensive (50-345x) Layer.provide (345x), Schedule retry (133x), Fiber spawn (52x). Layer reconstructs the DI context per call — provide once at startup, not per-request.
