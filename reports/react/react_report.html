The Corporate Open Source Model

React is a textbook example of corporate open-source: a project primarily developed by a single company (Meta) and shared with the world. Unlike community-driven projects like Effect-TS where a single maintainer is a crisis, React's multi-engineer corporate backing is a strength.

Healthy Bus Factor: No single point of failure 10 Meta engineers each contribute meaningfully. The top author (eps1lon at 23.4%) is replaceable within the team. Bus factor 80% ranges from 4 to 10 depending on the month. This is how corporate OSS should look.
